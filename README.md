# Simulador de Máquina de Turing

## Descripción
El Simulador de Máquina de Turing es una aplicación que permite visualizar y probar el funcionamiento de máquinas de Turing basadas en diversas expresiones regulares. Este programa cuenta con una interfaz gráfica que facilita la comprensión del proceso de ejecución de las máquinas de Turing, mostrando el estado de la cinta, el cabezal de lectura/escritura y el estado actual de la máquina.

## Requisitos
- Python 3.8 o superior
- Tkinter (generalmente incluido con Python)

## Instalación
Para instalar y ejecutar el simulador, se deben seguir los siguientes pasos:

1. Clonar el repositorio en la máquina local:
