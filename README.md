# Simulador de Máquina de Turing

Este proyecto es un **simulador de una Máquina de Turing** desarrollado en Python utilizando la librería **Tkinter** para la interfaz gráfica. El simulador permite a los usuarios interactuar con una máquina de Turing, procesar cadenas de caracteres conforme a diversas expresiones regulares, y observar el paso a paso del funcionamiento de la máquina de Turing.

## Requisitos

- **Python 3.x**.
- **Tkinter** (incluido en las instalaciones de Python).

## Descripción

El simulador permite al usuario:

- **Seleccionar una expresión regular** de una lista predefinida de máquinas de Turing.
- **Ingresar una cadena** de caracteres que será procesada por la máquina de Turing.
- Observar cómo la **cinta de la máquina de Turing** se llena con los caracteres de la cadena, y cómo el **cabezal** lee y escribe sobre ella.
- Controlar la simulación con los siguientes botones:
  - **Play**: Inicia la simulación de la máquina.
  - **Pause**: Pausa la simulación.
  - **Paso**: Avanza un paso de la simulación.
  - **Reset**: Restablece la máquina y la cadena ingresada.
- Ajustar la **velocidad de simulación** mediante una barra deslizante.
- **Ver el estado actual** de la máquina, el patrón de la expresión regular seleccionada y el alfabeto utilizado.

## Instalación

### Paso 1: Instalar Python

1. Descargar e instalar **Python 3.x** desde el [sitio oficial de Python](https://www.python.org/downloads/).
2. Asegurarse de seleccionar la opción **"Add Python to PATH"** durante la instalación.

### Paso 2: Clonar el Repositorio

1. Abrir una **IDE** (como **Visual Studio Code** o **PyCharm**).
2. En **Visual Studio Code**:
   - Abrir el terminal integrado con **Ctrl + `**.
   - Clonar el repositorio con el comando `git clone <URL del repositorio>`.

   Si no se usa **Git**, se puede descargar el archivo como un **zip** y extraerlo en una carpeta local.

### Paso 3: Abrir el Proyecto en una IDE

1. Abrir **Visual Studio Code** o **PyCharm**.
2. Seleccionar **Archivo > Abrir Carpeta** y abrir la carpeta donde se descargó el proyecto.

### Paso 4: Ejecutar el Proyecto

1. Abrir el archivo **`main.py`** dentro de la IDE.
2. Ejecutar el proyecto haciendo clic en el botón de **play** (triángulo verde) en la parte superior de la ventana de la IDE.

## Uso

1. **Ejecutar la Simulación**:
   - Después de abrir el proyecto en la IDE, hacer clic en el botón **Play** para comenzar la simulación.
2. **Interacción**:
   - **Expresión/MT**: Seleccionar una máquina de Turing de la lista desplegable, cada una asociada con una expresión regular diferente.
   - **Cadena**: Ingresar una cadena de caracteres para que sea procesada por la máquina de Turing.
   - **Play**: Inicia la ejecución de la máquina de Turing.
   - **Pause**: Pausa la simulación.
   - **Paso**: Avanza un paso de la simulación.
   - **Reset**: Restablece la máquina y la cadena ingresada.
   - **Velocidad**: Ajustar la velocidad de la simulación utilizando la barra deslizante.
